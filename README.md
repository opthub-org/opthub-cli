# OptHub CLI Client

This package provides a command-line interface to operate OptHub.


## Requirements

Python 2.7 or 3.5+


## Installation

```
$ pip install opthub-cli
```


## Usage

```
$ opt --help
```


## Documentation

https://opthub-org.github.io/opthub-cli/
